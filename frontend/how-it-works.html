Common Questions

Can this system be used in schools and universities? expand_more
Yes. Civic Ledger supports multi-level elections including schools, universities, associations, cooperatives, local councils, and national elections. Admins can create election categories, positions, and voter groups for each level.
Is my vote really secret and protected? expand_more
Yes. Your identity is verified before voting, but your ballot content is encrypted and stored separately. The system can verify that you voted once without exposing who you voted for.
Can an admin run more than one election at the same time? expand_more
Yes. Multiple elections can run in parallel for different groups or levels. For example, a university can run student union elections while a department runs class representative voting.
